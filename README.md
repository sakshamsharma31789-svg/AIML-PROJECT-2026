# AIML-PROJECT-2026
            
                     #TRIAGE-AI:SMART PATIENT PRIORITY CHEKCER

Triage-AI is an intelligent Probelm Solving Agent which is designed for emergency wards of the hospital. In medical environment , due to high inflow of patients the "First Come , First Served' methodolgy is inefficient. This project implements Utility Based Agent that calculates a real time priority score for patients based on sympthoms like current condition , severity of case , age in order to ensure that the patient with higher severity is treated first .

Concepts  USED-->
This project is made by using following concpets of AI:
Intelligent Agent: It operates as a Utility-Based Agent that acts rationally to maximize "expected utiltiy" or in simple terms lives saved.
Data Representaion: The project used pandas library's DataFrames to structure the data and information of patient
Knowledge Representaion: It uses medical knowleadge embedded in dictionay in order to weight symptoms effect.
Rational Decision Mking: It evaluates the state of the emergency room to provide the better action which can help patient's treatement.


Technicality used:
Language used: Python 3.14
Libraries used: pandas(for data representation),random
